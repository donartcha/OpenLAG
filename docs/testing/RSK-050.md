# Freeze Runtime Context Drift Risk

Represents the operational risk that global command invocation without project-local runtime prerequisites can produce export failures or inconsistent documentary outputs.

It exists in 0.5.0 because freeze evidence became release-relevant; runtime-context drift now has governance impact, not only developer inconvenience.
