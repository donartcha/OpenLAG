# Version-Scoped Freeze As Release Evidence

This decision establishes that documentary exports are part of lifecycle evidence, not optional narrative attachments.

It exists to solve audit ambiguity from baseline workflows where behavior was verifiable only through branch diffs and command logs.

The 0.5.0 impact is that freeze outputs are expected to be version-scoped, reproducible under controlled runtime conditions, and traceable back to root lifecycle artifacts.
